# Contributing

Yokohama Wind & Safety Now への貢献を歓迎します。

このプロジェクトは、横浜市の強風リスクを分かりやすく見える化し、外出判断や防災意識に役立てるための OSS です。初心者の方の Issue や Pull Request も歓迎します。

## 貢献の例

- バグ報告
- UI の改善
- スマホ表示の改善
- 防災メモの改善
- README の説明追加
- アクセシビリティ改善
- テスト追加
- Vercel デプロイ手順の改善

## 開発の始め方

```bash
npm install
npm run dev
```

確認:

```bash
npm run typecheck
npm run build
```

## Pull Request の流れ

1. Issue がある場合は、関連 Issue を確認します。
2. 変更内容を小さくまとめます。
3. `npm run typecheck` と `npm run build` を実行します。
4. Pull Request テンプレートに沿って説明を書きます。

## 大切にしたいこと

- 初心者にも読みやすいコードにする
- 防災情報として誤解されない表現にする
- 公式の避難情報ではないことを明示する
- スマホで見やすいことを優先する
- 地域の役に立つ改善を歓迎する
