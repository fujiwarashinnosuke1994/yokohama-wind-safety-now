# Code of Conduct

Yokohama Wind & Safety Now は、誰もが安心して参加できる OSS プロジェクトを目指します。

## 私たちの約束

参加者は、経験、年齢、国籍、性別、障害、職業、技術レベルにかかわらず、互いに敬意を持って接します。

## 望ましい行動

- 相手を尊重する
- 初心者にも分かりやすく説明する
- 建設的にフィードバックする
- 地域や防災に関わる情報を慎重に扱う
- 間違いに気づいたら丁寧に修正する

## 望ましくない行動

- 攻撃的な発言
- 差別的な表現
- ハラスメント
- 個人情報の公開
- 不安をあおる防災表現

## 適用範囲

この Code of Conduct は、Issue、Pull Request、Discussion、README、SNS など、このプロジェクトに関わるすべての場に適用されます。

## 報告

問題があった場合は、リポジトリ管理者に連絡してください。報告内容は必要な範囲で慎重に扱います。
