# Yokohama Wind & Safety Now

日本語名: 横浜 風速・防災ナウ

Yokohama Wind & Safety Now は、横浜市中心部の現在の風速・風向を表示し、強風時の外出、自転車、海沿い移動、子どもの送迎、車の運転判断に役立てるための防災・シビックテック系 OSS Web アプリです。

単なる天気アプリではなく、横浜市民・保護者・自転車利用者・ドライバー・観光客のための「強風リスクの見える化ツール」として作っています。

## 社会的意義

強風は、日常の移動や安全判断に直接関わります。特に横浜のように海沿い、橋、高低差、ビル風の影響を受けやすい地域では、同じ天気でも移動手段や場所によって体感リスクが変わります。

このアプリは、現在の風速を大きく表示し、安全レベルと防災メモを添えることで、次のような判断を助けます。

- 子どもの送迎をいつも通り行うか
- 自転車やベビーカーで移動するか
- 海沿い、橋の上、開けた道を避けるか
- 車のドア開閉や運転に注意すべきか
- 観光や屋外イベントの予定を見直すか

これは避難情報ではなく、外出判断の補助情報です。警報、避難情報、交通情報が必要な場合は、気象庁・横浜市・交通機関などの公式情報を確認してください。

## 主な機能

- 横浜市中心部の現在の風速を大きく表示
- 風向、取得時刻、データソースを表示
- Open-Meteo API を利用
- API キー不要
- 風速に応じた安全レベルを表示
- 防災メモを表示
- スマホで見やすいシンプル UI
- Vercel にデプロイしやすい Next.js 構成
- MIT License

## 安全レベル

| 風速 | 表示 | 目安 |
| --- | --- | --- |
| 0以上3未満 m/s | 穏やか | 通常の外出はしやすい |
| 3以上6未満 m/s | やや風あり | 自転車やベビーカーは少し注意 |
| 6以上10未満 m/s | 強め | 海沿い・橋の上・自転車移動に注意 |
| 10以上15未満 m/s | 強風 | 外出、送迎、自転車、車のドア開閉に注意 |
| 15以上 m/s | 危険 | 不要不急の外出を避ける目安 |

## 使用データソース

- データソース: Open-Meteo
- API: `https://api.open-meteo.com/v1/forecast?latitude=35.4437&longitude=139.6380&current=wind_speed_10m,wind_direction_10m&wind_speed_unit=ms&timezone=Asia%2FTokyo`
- 使用地点: 横浜市中心部
- latitude: `35.4437`
- longitude: `139.6380`

## セットアップ方法

前提:

- Node.js 20.9 以上
- npm

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 動作確認

```bash
npm run typecheck
npm run build
```

## Vercel デプロイ方法

1. このリポジトリを GitHub に push します。
2. Vercel にログインします。
3. New Project を選びます。
4. GitHub リポジトリを Import します。
5. Framework Preset が Next.js になっていることを確認します。
6. Deploy を押します。

環境変数は不要です。Open-Meteo は API キーなしで利用できます。
