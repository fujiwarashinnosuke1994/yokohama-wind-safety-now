# Yokohama Wind & Safety Now

日本語名: 横浜 風速・防災ナウ

Demo URL: https://yokohama-wind-safety-now.vercel.app

Yokohama Wind & Safety Now は、現在の横浜市中心部の風速・風向・過去60分程度の推移を、リアルタイムに近い形で確認できる防災・外出判断補助ツールです。強風時の外出、自転車移動、海沿い移動、お子様の送迎、車の運転判断、車のドア開閉時の注意、飛来物リスクの把握など、日常の安全判断を補助するための防災・シビックテック系 OSS Web アプリとして作っています。

現在は、風速・風向に加えて、横浜市の現在天気と8日後までの天気予報、つまり今日を含む9日間の予報も確認できます。通学・通塾・送迎・通勤・車の運転判断の補助として、天気、気温、体感温度、降水量、降水確率、最大風速、最大瞬間風速をまとめて表示します。

開発者: 株式会社数強塾 代表 藤原進之介

株式会社数強塾 代表 藤原進之介が、通学・送迎・外出判断の補助を目的として開発した防災・Civic Techアプリです。

単なる天気アプリではなく、横浜市民・保護者・自転車利用者・ドライバー・観光客のための「強風リスクの見える化ツール」として作っています。

## 社会的意義

強風は、日常の移動や安全判断に直接関わります。特に横浜のように海沿い、橋、高低差、ビル風の影響を受けやすい地域では、同じ天気でも移動手段や場所によって体感リスクが変わります。

このアプリは、現在の風速を大きく表示し、安全レベルと防災メモを添えることで、次のような判断を助けます。

- 子どもの送迎をいつも通り行うか
- 自転車やベビーカーで移動するか
- 海沿い、橋の上、開けた道を避けるか
- 車のドア開閉や運転に注意すべきか
- 車を運転するか、風で車体やハンドル操作に不安がないか
- 飛来物リスクや、物が飛んでくる危険が高まっていないか
- 観光や屋外イベントの予定を見直すか

ただし、このアプリは「運転できる」「安全である」と断定するものではありません。表示する風速やグラフはあくまで補助情報です。警報、避難情報、交通情報、道路状況、公共交通機関の運行状況が必要な場合は、気象庁・横浜市・交通機関・道路管理者などの公式情報を必ず確認してください。

## 防災・Civic Tech の観点

このプロジェクトは、防災、Civic Tech、Public Safety、Mobility Safety の観点から、地域の移動判断を支える小さな公共的ツールを目指しています。

- 防災: 強風時の危険を日常生活の言葉で見える化する
- Civic Tech: 横浜市民や来訪者が使える OSS として公開する
- Public Safety: 風による事故やヒヤリハットを減らすための判断材料を提供する
- Mobility Safety: 自転車、徒歩、送迎、車移動、海沿い移動のリスク確認を助ける

## 更新とグラフ

- 手動更新: 画面上の「最新の風速を取得」ボタンから、現在の風速・風向・取得時刻を再取得できます。
- 自動更新: 画面表示中は60秒ごとに最新データを取得します。
- 現在天気: Open-Meteo の `current` データを使い、天気、気温、体感温度、降水量、雲量、風速、風向、最大瞬間風速を表示します。
- 8日後までの予報: Open-Meteo の `daily` データを使い、今日を含む9日分の天気、気温、降水確率、降水量、風速、最大瞬間風速を表示します。
- 過去60分: Open-Meteo の `minutely_15` データを使い、棒グラフで表示します。
- 過去12時間: Open-Meteo の `hourly` データを使い、折れ線グラフで表示します。
- 過去30日: Open-Meteo Archive API の日別最大風速を使い、折れ線グラフで表示します。
- グラフ強調: 10m/s以上を強風注意域の目安、15m/s以上を危険域の目安として強調します。

観測値の更新頻度はデータ提供元に依存します。このアプリが60秒ごとに取得しても、Open-Meteo 側の観測値が同じ場合は画面上の風速が変わらないことがあります。

データの更新頻度・粒度は提供元に依存します。実際の危険性は周囲の状況、地形、建物、交通状況によって変わります。

過剰アクセスを避けるため、更新中の重複取得を抑え、非表示タブでは自動更新を行わない設計にしています。

## 安全レベル

| 風速 | 表示 | 目安 |
| --- | --- | --- |
| 0以上3未満 m/s | 通常 | 緑。比較的穏やかな風の目安 |
| 3以上6未満 m/s | やや風あり | 青。危険表示ではなく情報表示 |
| 6以上10未満 m/s | 注意 | 黄。海沿い・橋の上・自転車移動では注意 |
| 10以上15未満 m/s | 強風注意 | 橙。外出、送迎、自転車、車のドア開閉、飛来物リスクに注意 |
| 15以上20未満 m/s | 危険 | 赤。不要不急の外出を控える判断材料 |
| 20以上 m/s | 非常に危険 | 紫。公式情報と周囲の状況を必ず確認 |

## 使用データソース

- データソース: Open-Meteo
- API: `https://api.open-meteo.com/v1/forecast?latitude=35.4437&longitude=139.6380&current=temperature_2m,apparent_temperature,precipitation,rain,weather_code,cloud_cover,wind_speed_10m,wind_direction_10m,wind_gusts_10m&daily=weather_code,temperature_2m_max,temperature_2m_min,precipitation_probability_max,precipitation_sum,wind_speed_10m_max,wind_gusts_10m_max&wind_speed_unit=ms&timezone=Asia%2FTokyo&forecast_days=9`
- 現在天気: Open-Meteo Forecast API の `current`
- 8日後までの予報: Open-Meteo Forecast API の `daily`
- 過去12時間: Open-Meteo Forecast API の `hourly=wind_speed_10m`
- 過去30日: Open-Meteo Archive API の `daily=wind_speed_10m_max`
- 使用地点: 横浜市中心部
- latitude: `35.4437`
- longitude: `139.6380`

## データソースと限界

このアプリは Open-Meteo API を利用しています。APIキー不要で動作しますが、観測値や予報の更新頻度、過去データの粒度、取得できる期間、欠損値の有無はデータ提供元に依存します。

過去60分は15分単位のデータ、過去12時間は時間単位のデータ、過去30日は日別最大風速を利用します。そのため、グラフの細かさは期間によって異なります。

現在天気の降水確率は、当日予報の最大降水確率を表示します。リアルタイムの現在値そのものではないため、通学・通塾・送迎・通勤・車の運転判断では、最新の公式情報や周囲の状況も確認してください。

## 免責

このアプリは補助情報を提供するものであり、公式な避難情報・警報・交通情報の代替ではありません。「運転できる」「外出して安全である」と断定するものではありません。

必要に応じて、気象庁、横浜市、交通機関、道路管理者などの公式情報を確認してください。
