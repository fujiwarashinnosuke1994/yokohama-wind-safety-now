# Yokohama Wind & Safety Now

日本語名: 横浜 風速・防災ナウ

Demo URL: https://yokohama-wind-safety-now.vercel.app

Yokohama Wind & Safety Now は、現在の横浜市中心部の風速・風向・過去60分程度の推移を、リアルタイムに近い形で確認できる防災・外出判断補助ツールです。強風時の外出、自転車移動、海沿い移動、お子様の送迎、車の運転判断、車のドア開閉時の注意、飛来物リスクの把握など、日常の安全判断を補助するための防災・シビックテック系 OSS Web アプリとして作っています。

単なる天気アプリではなく、横浜市民・保護者・自転車利用者・ドライバー・観光客のための「強風リスクの見える化ツール」として作っています。

## 社会的意義

強風は、日常の移動や安全判断に直接関わります。特に横浜のように海沿い、橋、高低差、ビル風の影響を受けやすい地域では、同じ天気でも移動手段や場所によって体感リスクが変わります。

このアプリは、現在の風速を大きく表示し、安全レベルと防災メモを添えることで、次のような判断を助けます。

- 子どもの送迎をいつも通り行うか
- 自転車やベビーカーで移動するか
- 海沿い、橋の上、開けた道を避けるか
- 車のドア開閉や運転に注意すべきか
- 車を運転するか、風で車体やハンドル操作に不安がないか
- 飛来物リスクや、物が飛んでくる危険が高まっていないか
- 観光や屋外イベントの予定を見直すか

ただし、このアプリは「運転できる」「安全である」と断定するものではありません。表示する風速やグラフはあくまで補助情報です。警報、避難情報、交通情報、道路状況、公共交通機関の運行状況が必要な場合は、気象庁・横浜市・交通機関・道路管理者などの公式情報を必ず確認してください。

## 防災・Civic Tech の観点

このプロジェクトは、防災、Civic Tech、Public Safety、Mobility Safety の観点から、地域の移動判断を支える小さな公共的ツールを目指しています。

- 防災: 強風時の危険を日常生活の言葉で見える化する
- Civic Tech: 横浜市民や来訪者が使える OSS として公開する
- Public Safety: 風による事故やヒヤリハットを減らすための判断材料を提供する
- Mobility Safety: 自転車、徒歩、送迎、車移動、海沿い移動のリスク確認を助ける

## 主な機能

- 横浜市中心部の現在の風速を大きく表示
- 風向、取得時刻、データソースを表示
- Open-Meteo API を利用
- API キー不要
- 風速に応じた安全レベルを表示
- 「最新の風速を取得」ボタンによる手動更新
- 画面表示中に60秒ごとに最新データを自動取得
- 過去60分の風速推移を棒グラフで表示
- 10m/s以上の棒を赤系、15m/s以上の棒を濃い赤で表示
- 防災メモを表示
- スマホで見やすいシンプル UI
- Vercel にデプロイしやすい Next.js 構成
- MIT License

## 更新とグラフ

- 手動更新: 画面上の「最新の風速を取得」ボタンから、現在の風速・風向・取得時刻を再取得できます。
- 自動更新: 画面表示中は60秒ごとに最新データを取得します。
- 風速推移: Open-Meteo の `minutely_15` データを使い、過去60分程度の風速推移を棒グラフで表示します。
- グラフ色: 10m/s以上は赤系、15m/s以上はさらに濃い赤で表示します。

観測値の更新頻度はデータ提供元に依存します。このアプリが60秒ごとに取得しても、Open-Meteo 側の観測値が同じ場合は画面上の風速が変わらないことがあります。

赤色は強風域の目安です。実際の危険性は周囲の状況、地形、建物、交通状況によって変わります。

過剰アクセスを避けるため、更新中の重複取得を抑え、非表示タブでは自動更新を行わない設計にしています。

## 安全レベル

| 風速 | 表示 | 目安 |
| --- | --- | --- |
| 0以上3未満 m/s | 穏やか | 通常の外出はしやすい |
| 3以上6未満 m/s | やや風あり | 自転車やベビーカーは少し注意 |
| 6以上10未満 m/s | 強め | 海沿い・橋の上・自転車移動に注意 |
| 10以上15未満 m/s | 強風 | 外出、送迎、自転車、車のドア開閉に注意 |
| 15以上 m/s | 危険 | 不要不急の外出を避ける目安 |

## 使用データソース

- データソース: Open-Meteo
- API: `https://api.open-meteo.com/v1/forecast?latitude=35.4437&longitude=139.6380&current=wind_speed_10m,wind_direction_10m&minutely_15=wind_speed_10m&wind_speed_unit=ms&timezone=Asia%2FTokyo&past_minutely_15=4&forecast_minutely_15=1`
- 使用地点: 横浜市中心部
- latitude: `35.4437`
- longitude: `139.6380`

## セットアップ方法

前提:

- Node.js 20.9 以上
- npm

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 動作確認

```bash
npm run typecheck
npm run build
```

## Vercel デプロイ方法

1. このリポジトリを GitHub に push します。
2. Vercel にログインします。
3. New Project を選びます。
4. GitHub リポジトリを Import します。
5. Framework Preset が Next.js になっていることを確認します。
6. Deploy を押します。

環境変数は不要です。Open-Meteo は API キーなしで利用できます。
